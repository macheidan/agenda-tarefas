# agenda-tarefas

Projeto React + Vite, hospedado no GitHub Pages (branch `gh-pages`).

## Deploy

O site é servido a partir da branch `gh-pages`, gerada por `npm run deploy`
(que roda `vite build` e publica `dist/` via `gh-pages`).

Após mudanças em código que afetem a UI, **rodar `npm run deploy` automaticamente**
sem pedir confirmação — o usuário autorizou deploys automáticos.
